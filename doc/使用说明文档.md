README

## 一、环境依赖
- Node.js v20.11.1+
- Vue3 v3.4.21
- Vite v5.2.0

## 二、项目部署
### 1. 安装Node环境
[node.js](https://nodejs.org/en/)
### 2. 依赖安装
```bash
npm install
```
### 3. 项目打包
```bash
npm run build
```
打开./dist目录下的index.html文件即可。
